# Express.js Tutorial Project

Project ini adalah **pembelajaran dasar Express.js** menggunakan **Node.js v20**.
Struktur project dibuat sederhana agar mudah dipahami dan akan **dikembangkan secara bertahap**.

---

## Tech Stack

Project ini dibangun menggunakan teknologi berikut:

- **Node.js v20**
  Runtime environment untuk menjalankan JavaScript di server.
  https://nodejs.org/

- **Express.js 5.2.1**
  Web framework minimalis untuk membuat server dan API dengan Node.js.
  https://expressjs.com/

- **Body Parser 2.2.2**
  Middleware untuk membaca request body seperti JSON dan form data.
  https://www.npmjs.com/package/body-parser

- **Nodemon 3.1.14**
  Development tool untuk otomatis restart server saat file berubah.
  https://nodemon.io/

- **Pug ^3.0.3**
  Development tool untuk Render Data Dinamis.
  https://pugjs.org/

## Node Version

Project ini menggunakan:

```
Node.js v20
```

Cek versi Node:

```bash
node -v
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Server

Menjalankan server:

```bash
npm start
```

Server akan berjalan di:

```
http://localhost:3000
```
